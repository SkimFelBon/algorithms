# most used git commands
## status
git status

## stage changes
git add <filename>

## simply check changes
git diff <filename>

## show staged changes
git diff --staged/ git diff --cached
git diff --staged <filename>

## commit changes
git commit

## to publish your local commits
git push

## unstage changes
git reset

## discard changes
git checkout <somedir> OR <file>

## to update your local branch
git pull
